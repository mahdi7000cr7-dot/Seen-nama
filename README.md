# Seen-nama
🎬 پروژه وب‌سایت نمایش فیلم و سریال (Movie Streaming Website)

این پروژه یک وب‌سایت نمایش فیلم و سریال است که با استفاده از HTML، CSS و JavaScript ساخته شده و از APIهای خارجی برای دریافت اطلاعات فیلم‌ها استفاده می‌کند. هدف پروژه، شبیه‌سازی یک پلتفرم ساده مشابه سرویس‌های استریم فیلم (مثل Netflix) برای پروژه دانشگاهی است.

✨ ویژگی‌ها 🎥 نمایش فیلم‌ها و سریال‌های جدید ⭐ بخش Featured (فیلم‌های ویژه) 🔍 سیستم اسلایدر حرفه‌ای برای نمایش محتوا 📦 استفاده از API برای دریافت اطلاعات فیلم‌ها 💾 سیستم کش (Cache) برای کاهش درخواست‌های API و افزایش سرعت ⚡ بارگذاری سریع با استفاده از localStorage 📱 طراحی ریسپانسیو (Responsive) 🧭 ناوبری بین صفحات (لیست کامل + صفحه پخش فیلم) 🧠 تکنولوژی‌ها HTML5 CSS3 JavaScript (Vanilla JS) Splide.js (برای اسلایدرها) Watchmode API (برای فیلم و سریال) OMDb API localStorage (برای کش داده‌ها)

⚙️ نحوه کار پروژه 📡 دریافت داده‌ها از API

پروژه از Watchmode API و OMDb API برای دریافت اطلاعات فیلم‌ها استفاده می‌کند.

💾 سیستم کش (Cache)

در فایل api.js یک سیستم کش هوشمند وجود دارد:

اگر داده قبلاً ذخیره شده باشد و کمتر از 24 ساعت گذشته باشد → از localStorage خوانده می‌شود در غیر این صورت → درخواست جدید به API ارسال می‌شود اگر داده معتبر نباشد → به صورت ایمن رد می‌شود 🎞️ سیستم نمایش فیلم‌ها

در فایل MovieLists.js:

فیلم‌ها به صورت اسلایدر نمایش داده می‌شوند هر کارت شامل: پوستر فیلم عنوان فیلم با کلیک روی هر فیلم → انتقال به صفحه پخش (play.html) 🎠 اسلایدرها

با استفاده از Splide.js:

نمایش چندین فیلم در یک ردیف پشتیبانی از موبایل و دسکتاپ قابلیت drag و swipe
